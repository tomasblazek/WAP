WAP - Výběr zboží návod

Obecné informace

Tento html dokument slouží jako návod k použití javaskriptového modulu filter.js pro výběr zboží do předmětu Internetové aplikace (WAP). Demo html s použitím zde zde.

Návod k použití

  1. Do html dokumentu, kde má být knihovna pro výběr zboží využávána je nutné vložit referenci na Javaskriptový soubor s knihovou. <script src='filter.js'></script>
  2. Vzhled filtru je definován v souboru filter.css ve složce css a ten je tedy taky zapotřebí vložit na načátku html dokumentu. <link rel='stylesheet' type='text/css' href='css/filter.css'>
    Obecný vzhled demo příkladu a tohoto návodu je definován v souboru general.css ve složce css. <link rel='stylesheet' type='text/css' href='css/general.css'>
  3. Dále je nutné vložit identifikátor se jménem filtru do bloku (např. div), kde chceme provádět vyhledávání. <div id='my-fitler1'>
  4. Každý blok v html souboru reprezentující prvek, který má být součástí vyhledávací množiny označit třídou s klíčovým slovem item. Uvnitř jednotlivých prvků je možné vkládat jednotlivé atributy, u kterých je nutné uvést třídu se jménem atributu, se kterou může filtr pracovat.
  5. Konfigurační objekt pro vytvoření vlastního filtru má tento tvar. 
    var myFitler1Structure = {
	"name": {
		"label": "Název",
		"type": "text",
		},
	"size": {
		"labelFrom": "Od",
		"labelTo": "Do",
		"type": "number",
	},
	 "type": {
		"label": "Typ",
		"type": "select",
	},
};
    Každá položka na první úrovni značí atribut, dle kterého se bude vyhledávat. Káždy atribut musí mít ještě dva údaje. A to jsou název návěští (label) a typ pole (text, number, select).
  6. Nakonec zbýva v html dokumentu ještě zavolat inicializační funkci filtru z knihovny. K tomu je možné použít například funkci onload, která inicializuje filter po načtení okna. window.onload = setupFilter("my-fitler1", myFitler1Structure);
